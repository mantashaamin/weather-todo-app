# 📝 To-Do App

A modern To-Do application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add, edit, and delete tasks
- Priority tagging (Low, Medium, High)
- Emoji support 🔥✅💪
- Task completion tracking
- Local storage (data saved even after refresh)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
