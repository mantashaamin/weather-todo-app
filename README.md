# 🚀 Productivity Dashboard (To-Do + Weather App)

A simple yet functional **Productivity Dashboard** built during my internship at Cognifyz Technologies.  
This project combines a **To-Do App** and a **Weather App** into a single interface.

---

## 🌐 Live Demo
👉 https://mantashaamin.github.io/weather-todo-app/

---

## 📌 About the Project

During the internship, problem statements were provided instead of fixed projects.  
Based on those requirements, I designed and built this dashboard by combining:

- 📝 Task management (To-Do App)
- 🌦️ Real-time weather data (Weather App)

---

## ✨ Features

### 📝 To-Do App
- Add tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Active / Completed)
- Data stored using Local Storage

---

### 🌦️ Weather App
- Search weather by city
- Displays:
  - 🌡 Temperature
  - ☁️ Condition
  - 💧 Humidity
  - 🌬 Wind speed
  - 🕒 Local time
- Uses real-time API data

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- OpenWeather API  
- Local Storage
